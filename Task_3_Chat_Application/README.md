# Task 3: Chat Application

## Description
This is a simple text-based chat application using Python.
It follows a client-server model where two users can communicate in real time
through the command line.

## Technologies Used
- Python
- Socket Programming

## Files
- server.py – Starts the server and waits for client connection
- client.py – Connects to the server and sends messages

## How to Run

1. Open Command Prompt
2. Navigate to the Task_3_Chat_Application folder

### Run Server
python server.py

### Run Client (in another Command Prompt)
python client.py

## Sample Output

Client connected

Client:
Connected to server
