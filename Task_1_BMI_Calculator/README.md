## Description
This is a simple command-line BMI (Body Mass Index) Calculator written in Python.
The program takes user input for weight and height, calculates BMI, and displays
the BMI category.

## How It Works
1. User enters weight in kilograms
2. User enters height in meters
3. Program calculates BMI using the formula:
   BMI = weight / (height * height)
4. Program displays BMI value and category

- Python
